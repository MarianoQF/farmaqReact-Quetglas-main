Farmaq Ecommerce app es un proyecto para un emprendimiento local en Corrientes Capital-Argentina. Es una pyme que comercializa y distribuye productos de farmacia. Tome las bases visuales de una landing page antes creada en un curso anterior.

La app fue realizada durante el curso de React de Coderhouse, siguiendo las directívas, consejos y enseñanzas del profesor Felix Blanco.

# Documentación

## Librería

React-router-dom

## Base de datos

Firebas(fireStore para la base de datos, es una base de datos noSQL, proveeída por Google)

## Imagenes

Las imagenes de los productos, iconos y logos de la marca estan en la carpeta imgs, la cual se encuentra dentro de la carpeta public.
la ruta para utilizar dichas imagenes sería:
'/public/imgs/...'

## SRC

Dentro de la carpeta src, podemos encontrar la carpeta components que tiene varias sub carpetas con archivos jsx y css para la estructura y el estilado.
También cuenta con una carpeta context y service, ademas de los archivos app e index.
